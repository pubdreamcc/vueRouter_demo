## 实现方法

1. 搭建vue-cli项目（不会自行百度）;
2. 新建Main,Person,User三个单组件文件;
3. 配置路由的匹配规则(router/index.js);
4. 主组件文件设置导航链接和路由渲染容器router-view;
5. 修改main.js 中vue实例router属性
